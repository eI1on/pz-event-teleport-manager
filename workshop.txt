title=[B41] Event Teleport Manager
description=This mod provides a comprehensive event management system that allows [b]server administrators[/b] to create events through an intuitive UI interface, enabling [b]players to register and teleport[/b] to event locations during designated time windows.
description=
description=[h1]Core Features[/h1]
description=[b]Event Creation & Management:[/b]
description=[list]
description=[*]Create events with custom names and descriptions through the admin UI
description=[*]Set specific coordinates for event locations on the map
description=[*]Configure teleport time windows (when players can teleport to the event area)
description=[*]Unique event IDs with 6-character codes for easy identification
description=[*]Real-time event status tracking and management
description=[/list]
description=
description=[b]Player Registration System:[/b]
description=[list]
description=[*]Players can view available events and register through the client UI
description=[*]Admin-controlled registration options (self-registration or admin-only)
description=[*]Bulk player management for efficient event coordination
description=[*]Clear visibility of event details and registration status
description=[/list]
description=
description=[b]Teleportation System:[/b]
description=[list]
description=[*]Time-windowed teleportation access (only during active teleport windows)
description=[*]Automatic position protection - saves player locations before teleporting
description=[*]Multiple return options: original position or safehouse after events
description=[*]Safety checks including zombie density validation before teleporting
description=[*]Configurable teleport cooldown to prevent abuse
description=[/list]
description=
description=[h1]How It Works[/h1]
description=[b]For Players:[/b]
description=[list]
description=[*]Access the Event Panel from the left side of your screen
description=[*]Browse available events and see teleport time windows
description=[*]Register for events you want to participate in
description=[*]Teleport to event locations during the designated time windows
description=[*]Return to your original position or safehouse when done
description=[/list]
description=
description=[b]For Administrators:[/b]
description=[list]
description=[*]Use the Admin Panel to create and manage events
description=[*]Set event locations, names and teleport time windows
description=[*]Manage player registrations and monitor participation
description=[*]Control who can access events and when teleportation is available
description=[/list]
description=
description=[h1]Key Concepts[/h1]
description=[b]Event Creation:[/b] Admins create events through the UI, setting locations and teleport time windows
description=[b]Registration:[/b] Players can see events and register to participate
description=[b]Teleport Windows:[/b] Time periods when registered players can teleport to event locations
description=[b]Position Protection:[/b] Player locations are automatically saved and restored
description=
description=[h1]Configuration Options[/h1]
description=Customize the mod behavior through sandbox settings:
description=[list]
description=[*][b]Event Admin Access:[/b] Set minimum permission level (None to Admin)
description=[*][b]Player Registration:[/b] Enable/disable self-registration
description=[*][b]Teleport Return Mode:[/b] Choose return behavior (Original, Safehouse, or Both)
description=[*][b]Zombie Check:[/b] Enable safety checks before teleporting
description=[*][b]Zombie Threshold:[/b] Maximum zombie count for safe teleportation (0-50)
description=[*][b]Teleport Cooldown:[/b] Delay between teleports (0-3600 seconds)
description=[*][b]Position Protection:[/b] Automatically save player positions
description=[/list]
description=
description=[h1]Safety & Security Features[/h1]
description=[list]
description=[*]Automatic position saving before teleportation
description=[*]Zombie density validation to prevent dangerous teleports
description=[*]Permission-based access control system
description=[*]Teleport cooldown to prevent spam/abuse
description=[*]Event status validation before allowing actions
description=[*]Comprehensive logging for admin oversight
description=[/list]
description=
description=[h1]User Interface[/h1]
description=[list]
description=[*]Intuitive dual-interface design for admins and players
description=[*]Real-time event status updates and registration management
description=[*]Clear teleport time window indicators
description=[*]Responsive notifications and status messages
description=[*]Easy bulk operations for efficient event management
description=[/list]
description=
description=[h1]Multiplayer Optimized[/h1]
description=[list]
description=[*]Server-side event management with client synchronization
description=[*]Automatic updates pushed to all connected players
