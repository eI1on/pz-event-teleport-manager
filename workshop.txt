title=[B41] Event Teleport Manager [DEV Branch]
description=[h1]Event Teleport Manager [DEV Branch][/h1]
description=This mod introduces a comprehensive event management system, allowing [b]server administrators[/b] to create scheduled events with specific times and locations while enabling [b]players to register and participate[/b].
description=
description=[h1]How to Access the Event Teleport Manager[/h1]
description=[b]For All Players:[/b]
description=[list]
description=[*] From the Client Panel on the left side of the screen. Event Teleport Manager button
description=[*] Access available events, registration status, and teleportation options.
description=[/list]
description=
description=[b]Admin Features:[/b]
description=[list]
description=[*] Create, edit, and manage events through the comprehensive admin interface found in the Admin Panel, or likewise, the Client Panel.
description=[*] Set event schedules, locations, and manage player registrations.
description=[*] Teleport players to events and return them to original positions.
description=[/list]
description=
description=[h1]Features[/h1]
description=The mod provides a complete event management solution with extensive customization options:
description=[list]
description=[*] Create events with custom names, descriptions, and specific start/end times when teleportation is available.
description=[*] Set precise event locations with coordinate-based positioning.
description=[*] Player registration system with manual admin registration or open registration modes.
description=[*] Automatic event status tracking (Future, Active, Ended).
description=[*] Teleportation to event locations with original position saving.
description=[*] Bulk player management - teleport multiple participants simultaneously.
description=[*] Real-time event notifications when events start or end.
description=[*] Intuitive dual-interface design for both administrators and players.
description=[*] Permission-based access control with configurable admin levels (from sandbox options).
description=[/list]
description=
description=[h1]Event Management System[/h1]
description=The mod features a sophisticated event lifecycle management system:
description=
description=[h2]1. Event Creation & Scheduling[/h2]
description=Administrators can create detailed events with precise timing which converts to client timezone.
description=[b]Options include:[/b]
description=[list]
description=[*][b]Event Details:[/b] Custom names, descriptions, and unique event IDs.
description=[*][b]Time Management:[/b] Set start and end times with timezone support.
description=[*][b]Location Setting:[/b] Define exact coordinates or use current position.
description=[*][b]Status Tracking:[/b] Automatic status updates based on current time.
description=[/list]
description=
description=[h2]2. Player Registration[/h2]
description=Flexible registration system accommodating different server needs.
description=[b]Features include:[/b]
description=[list]
description=[*][b]Self-Registration:[/b] Players can register themselves for events (configurable from sandbox options).
description=[*][b]Admin Registration:[/b] Administrators can add/remove players manually.
description=[*][b]Registration Status:[/b] Clear indication of registration status per event.
description=[*][b]Bulk Management:[/b] Add multiple players simultaneously.
description=[/list]
description=
description=[h2]3. Teleportation System[/h2]
description=Reliable teleportation with position preservation.
description=[b]Capabilities include:[/b]
description=[list]
description=[*][b]Event Teleportation:[/b] Transport players to active events instantly.
description=[*][b]Return System:[/b] Restore players to their original positions after events.
description=[*][b]Admin Controls:[/b] Teleport individual players or entire groups.
description=[*][b]Safety Checks:[/b] Validates event status and registration before teleporting.
description=[/list]
description=
description=[h1]Configuration Options[/h1]
description=Server administrators can customize the mod behavior through sandbox variables:
description=[list]
description=[*][b]Player Registration:[/b] Enable/disable self-registration for players.
description=[*][b]Admin Access Level:[/b] Set minimum permission level required for event management.
description=[/list]
description=
